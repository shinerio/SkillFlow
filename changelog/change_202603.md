## GitHub Release 自动发布与应用自动更新

date: 2026-03-06

### 新增功能
- **版本注入**：新增 `version.go`，通过 `-ldflags "-X main.Version=vX.Y.Z"` 在 CI 构建时注入版本号，本地开发默认为 `"dev"`
- **应用更新检测后端**：新增 `app_update.go`，包含：
  - `GetAppVersion()`：返回当前版本字符串
  - `CheckAppUpdate()`：查询 GitHub Releases API，按平台（windows/macos-intel/macos-apple-silicon）匹配下载链接，返回 `AppUpdateInfo`
  - `DownloadAppUpdate(url)`：异步下载新版本到临时目录，通过事件上报进度
  - `ApplyAppUpdate()`：Windows 专用，写 bat 脚本实现退出后 exe 替换并重启
  - `checkAppUpdateOnStartup()`：启动时静默检测（Version == "dev" 时跳过）
- **新增事件类型**：`core/notify/model.go` 增加 `app.update.available`、`app.update.download.done`、`app.update.download.fail`
- **前端更新横幅**：`App.tsx` 新增固定顶部横幅，四态状态机（available → downloading → ready_to_restart / download_failed），Windows 支持一键下载+重启，macOS 跳转发布页
- **CI 自动发布**：`.github/workflows/build.yml` 新增 `release` job，构建时注入版本号，macOS 产物打包为 zip，Windows 直接上传 exe，通过 `softprops/action-gh-release` 自动创建 GitHub Release

### 更新文件
- `app.go`：startup 中新增 `go a.checkAppUpdateOnStartup()`
- `frontend/wailsjs/`：重新生成 TypeScript 绑定（含 `AppUpdateInfo` 类型）
- `feature.md` / `feature_zh.md` / `README.md` / `README_zh.md`：同步新增应用自动更新功能文档

## SkillFlow 全量实现：后端 + 前端 + CI 构建

date: 2026-03-04

### 后端（Go 核心库）
- **数据模型**：`Skill`（含 SourceType/SourceSHA/LatestSHA）、`AppConfig`（Tools/Cloud）、`EventType` 通知常量
- **通知 Hub**：基于 channel 的线程安全 pub/sub，支持多订阅者
- **配置服务**：跨平台 `AppDataDir()`，5 个内置工具默认路径，JSON 持久化
- **注册表（Registry）**：全局 Installer/ToolAdapter/CloudProvider 插件注册
- **Skill 管理**：Validator（skill.md 校验）+ Storage（UUID/JSON meta sidecar，支持分类、重命名、删除、覆盖）
- **GitHub 安装器**：Scan/DownloadTo/GetLatestSHA，支持 httptest mock
- **本地安装器**：基于 Validator 扫描本地目录
- **同步适配器**：FilesystemAdapter（Push 展开分类目录，Pull 扫描有效 skill）
- **云备份**：Aliyun OSS / 腾讯 COS / 华为 OBS 三家 Provider，含 Sync/Restore/List
- **更新检测**：GitHub Commits API SHA 比对，空 URL 边界保护
- **Wails App 层**：24+ 方法绑定（Skills/Install/Sync/Config/Backup/Update），hub 事件转发

### 前端（React 18 + TypeScript + Tailwind CSS）
- **布局**：BrowserRouter 侧边栏导航，5 个路由页面
- **Dashboard**：搜索、拖拽分类、右键菜单、GitHub 安装对话框、窗口级拖拽导入
- **SyncPush**：按工具/范围（全部/分类/手动）推送，含冲突对话框
- **SyncPull**：扫描工具目录，批量导入，含冲突对话框
- **Backup**：实时事件进度显示，云端文件列表，一键备份/恢复
- **Settings**：工具路径管理（含自定义工具增删）、动态云凭证表单、通用配置

### CI/CD
- GitHub Actions 矩阵构建：macOS Intel + Apple Silicon + Windows，按 `v*` tag 触发

### 测试
- 21 条 Go 单元测试全部通过（config/install/notify/skill/sync/update 6 个包）
